GTK-Wcalc

The GTK+ frontend to Wcalc provides a simple and easy to use standalone program for analysis and synthesis. The GTK+ frontend runs on most major Unix-like operating systems as well as WIN32. Gtk-wcalc was developed using gtk+-1.2.10, but seems to compile and run with newer versions of gtk (gtk+-2.4.13 was verified to work).
