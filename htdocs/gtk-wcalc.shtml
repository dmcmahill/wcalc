GTK-Wcalc

The GTK+ frontend to Wcalc provides a simple and easy to use standalone program for analysis and synthesis. The GTK+ frontend runs on most major Unix-like operating systems as well as WIN32. Gtk-wcalc is primarily developed using gtk+-2.10.0 and newer but it should build with older versions in the 2.x series as well as 1.2.x The following screen shots (which are probably out of date )show examples of the Wcalc GTK+ interface.
