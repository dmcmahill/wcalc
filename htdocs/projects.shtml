Current Wcalc Projects

In Progress

  • Bug hunting. This is always a high priority.
  • Model verification. This is also high priority. When possible I try to collect data points from other tools for comparison.
  • Better gtk2 support. The development has went on for so long that gtk2 is really the prefered gtk platform, however gtk-wcalc still uses some obsolete gtk1 code.

Future

  • i18n. Some of the framework is in place as well as the start of a Spanish language translation. I have not worked on CGI i18n. In fact I'm not sure of the best way to proceed there.
  • I would like to figure out how to use a field solver to provide accuracy validation. I have tried to use MMTL for comparison but haven't fully learned to use that tool yet. Ideally I'd like to have some automated scripts to plot things like impedance and velocity as a function of some input parameter with both wcalc and MMTL to show how they compare.
  • More models.
    • co-planar waveguide
    • various microstrip discontinuities
    • spiral inductors. The ability to interface to fasthenry is being considered.
  • ability to print under WIN32
