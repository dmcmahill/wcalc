How to Get Wcalc

Source Format

Wcalc comes with complete source code which can easily be compiled on most Unix-like operating systems with no modifications. In addition, it should be possible to compile wcalc under WIN32 using the MinGW toolchain with the cygwin environment.

See the Files section on the Wcalc Sourceforge Project Page for the available downloads.

Binaries

Currently Wcalc is not distributed in binary format although it is possible that a WIN32 binary will be made available in the future.

Native Packages

Wcalc is also available as a native package for several operating systems. Please note that these are maintained by groups other than the Wcalc maintainer:

  • NetBSD: Wcalc Package.
  • NetBSD: All CAD Packages.

Other

It is anticipated that Wcalc will be included on the gEDA Suite CD-ROM begining with the CD-ROM release after gEDA version 20050830. Please note that the CD-ROM is maintained by someone other than the Wcalc maintainer.
